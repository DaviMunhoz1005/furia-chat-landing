## Resumo do projeto
Esta landing page foi desenvolvida como parte de um Challenge para o processo seletivo da FURIA. O objetivo deste projeto é criar uma página simples, porém eficaz, para promover o FURIA Telegram Bot, um chatbot interativo no Telegram que mantém os fãs do time FURIA CS:GO atualizados sobre as partidas e outras curiosidades.

A página contém informações sobre o bot, suas funcionalidades e oferece um botão para que os usuários possam acessar diretamente o bot no Telegram.

## Hospedagem

Este projeto está hospedado na plataforma Vercel. A Vercel oferece hospedagem gratuita e automática para projetos front-end, como esta landing page. Sempre que o repositório for atualizado, o deploy será feito automaticamente. A página está disponível 24/7, com alta disponibilidade e acesso imediato.

Você pode acessar a página através do seguinte link:
[Acesse a Landing Page do FURIA CS:GO Bot](https://furia-chat-landing.vercel.app/)

## 🔨 Funcionalidades do projeto

- Informações sobre o bot: Descrição do FURIA Telegram Bot, com ênfase em como ele mantém os fãs atualizados sobre as partidas e interações com o time.
- Botão de redirecionamento para o Telegram: Um botão proeminente na página que leva os usuários diretamente ao bot no Telegram para interagir com ele.
